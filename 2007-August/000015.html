Are you using PCLINUSOS or PCLINUXOS (from www.pclinuxos.org)?

Anyhow, the way to proceed with that kind of problems is to look for the
appropriate package.
Try this link: http://www.google.es/search?q=PCLINUxOS+libXext&meta=

You will find the package where the libxext library is included in pclinuxos.
:-)
Just in case it helps, in Debian, the needed package is "libxext-dev"

Install it and configure again :-)
