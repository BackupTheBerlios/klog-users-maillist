(Versión en español al final)

 I am  happy to announce that the release of 0.4.3 version of KLog is
 available.

This release is an urgent release because I have detected and fixed a 
bug causing data lost. It is recommended to update your KLog 
installation before continue using it.

The mailing list of KLog is in Klog-users at lists.berlios.de for 
discussion about KLog, wishlist, bug report, ideas, suggestions tips on 
using, ...

 The main language of the list if English.

 Subscribe at:

                 https://lists.berlios.de/mailman/listinfo/klog-users

 From the Changelog:

12-jan-2008 - 0.4.3
- BUG fixed: KLog copied the QSL sent date into the QSL rec date. Only 
QSL rec date was lost but not the QSL status.
- BUG fixed: KLog now recognises a "/J" (Jamboree On The Air) Scout 
Station.
- GUI improved: Added the cty.dat URL.


 KLog can still have many bugs and other not yet implemented features.
 You can download KLog from:

 http://jaime.robles.nu/klog

 You can see some screenshots in the web.

 The KLog's new-releases email list is created using the addresses of
 people I have previously received emails from about KLog. Feel free
 to ask me to remove your address if you don't want to be alerted of
 new KLog releases.

 Thanks for your help and I hope you find KLog useful.
 73

12-ene-2008 - 0.4.3

- Error corregido: KLog reconoce las estaciones scout "/J" (Jamboree On 
The Air).
- Interfaz de usuario: Añadida la URL del fichero cty.dat.
