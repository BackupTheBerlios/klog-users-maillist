 I am  happy to announce that the release of 0.5.6 version of KLog is
available.

The mailing list of KLog is in  Klog-users at lists.berlios.de for discussion
about KLog, wishlist, bug report, ideas, suggestions tips on using, ...

 The main language of the list if English.

 Subscribe at:

                  https://lists.berlios.de/mailman/listinfo/klog-users

We also have a list for development:

                  https://lists.berlios.de/mailman/listinfo/klog-devel

November-2010 0.5.7

- New feature: Added the Export needed QSL that allows to create an ADIF
file with all the QSO with new ones and new bands still not confirmed that
has not been QSLed.
  The objective is to export it to a QSL or label printing software (until
KLog implements that feature).
- Updated translations: SV by SM4ABE.
- GUI improved: Added sliders to be able to move panels.
- GUI improved: Changing the band combobox changes the TX Freq box and
viceversa.
- Minor fix: The PROGRAMVERSION tag in log was not properly formated.

 KLog may have many bugs.

 You can download KLog from::

        http://jaime.robles.es/klog


 You will find the tar.gz sources and the debian package.


 You can see some screenshots in the web.



 The KLog's new-releases email list is created using the addresses of
people I have previously received emails from about KLog. Feel free to
ask me to remove your address if you don't want to be alerted of new KLog
releases.



 Thanks for your help and I hope you find KLog useful.

 73

 Del fichero Changelog vemos los cambios:

Noviembre-2010 0.5.7

- Nueva funcionalidad: Añadida una herramienta de para exportar un fichero
ADIF con todos los QSO que son entidades nuevas o nuevas bandas aun no
confirmadas para los que no se ha enviado la QSL.
El objetivo es exportar a un programa de impresiónde QSL o etiquetas
(hasta que KLog implemente esa funcionalidad)
- Traducción actualizada: SV por SM4ABE.
- Interfaz gráfica mejorada: Añadidos deslizadores para poder mover los
paneles.
- Interfaz gráfica mejorada: El cambio de banda modifica la frecuencia de
TX y viceversa.
- Pequeña corrección: La etiqueta PROGRAMVERSION no se formateaba
correctamente.

KLog está disponible en inglés, español, portugués, portugués brasileño,
alemán, serbio, gallego y sueco.
