 I am  happy to announce that the release of 0.4.2 version of KLog is
 available.

The mailing list of KLog is in Klog-users at lists.berlios.de for 
discussion about KLog, wishlist, bug report, ideas, suggestions tips on 
using,
 ...

 The main language of the list if English.

 Subscribe at:

                 https://lists.berlios.de/mailman/listinfo/klog-users

 From the Changelog:

12-Sep-2007 - 0.4.2

- New feature: The user can now configure to require or not the 
mandatory data for each QSO (QRZ, date, time, band, mode, RST tx & RST 
rx. (if not all the mandatory fields are entered, KLog's behaviour may 
be unexpected but it allows to import logs from other software that 
does not add all the data).
- New feature: Klog creates a temp file (~.klog/tempklog.adi) where it 
saves automatically all the QSOs entered until the log is manually 
saved. It prevents the log to be lost in case of unprevented 
KLog/computer failure. (TNX SM5OUU)
- BUG fixed: If the field (maybe any other field too) COMMENT is more 
than one line long, the log is broken and cannot be read. (TNX SM5OUU)
- BUG fixed: KLog does not ask for the file name if you have previously 
opened one file and the name has not changed.
- BUG fixed: In the GUI, the "Preferences" is correctly shown and not 
as "&Preferences".
- BUG fixed: When modifying a QSO, the number of QSOs was incremented 
when the user click over OK.
- BUG fixed: If a QSO was deleted the awards/number of QSO where not 
decresed.
- GUI improved: Deleted the WAE box (the WAE calculations was not 
implemented).
- New translation: KLog is now also in Swedish. (TNX SM4BE)
- New translation: KLog is now also in Galician. (TNX Fuco Mera)
- Updated German translation. (TNX DL5PD)
- Updated Spanish translation.  (EA4TV).

This release of KLog isin English, Spanish, Portuguese, Brazilian 
portuguese, German, Serbian, Galizian and Swedish.

 KLog can still have many bugs and other not yet implemented features.
 You can download KLog from:

 http://jaime.robles.nu/klog

 You will find the tar.gz sources and the debian package (debian
 package only in my homepage)
 You can see some screenshots in the web.

 The KLog's new-releases email list is created using the addresses of
 people I have previously received emails from about KLog. Feel free
 to ask me to remove your address if you don't want to be alerted of
 new KLog releases.

 Thanks for your help and I hope you find KLog useful.
 73

 Del fichero Changelog vemos los cambios:

- Nueva funcionalidad: El usuario puede configurar el requerir o no los 
campos obligatorios para cada QSO (QRZ, fecha, hora, banda, modo, RST 
(rx y tx). (Si no se introducen todos los campos obligatorios, KLog 
puede tener un comportamiento inesperado, pero permite importar datos 
de programas que no incluyen todos estos campos).
- Nueva funcionalidad: Klog crea un fichero temporal que se guarda 
automáticamente guardando todos los QSO hasta que el log se guarda de 
forma manual. Previene de pérdida de datos en caso de fallo de KLog o 
el ordenador. (TNX SM5OUU)
- Error corregido: En el campo (puede que en otros tambien) COMENTARIO, 
si había más de una línea, el log se corrompía y no podía leerse. (TNX 
SM5OUU)
- Error corregido: KLog volvía a preguntar el nombre del fichero en el 
que guardar el log aunque no fuera nuevo y lo hubieramos abierto 
previamente
- Error corregido: En la interfaz gráfica, las "Preferencias" se 
muestran ahora correctamente y no como "&Preferencias".
- Error corregido: Cuando se modificaba un QSO, se incrementaba el 
número de QSO.
- Error corregido: Si se borraba un QSO, no se decrementaba el número de 
QSO y diplomas.
- Interfaz gráfica mejorada: Se ha eliminado el campo WAE (porque no 
estaban implementada la gestión de dicho diploma).
- Nueva traducción: KLog está ahora también en sueco. (TNX SM4BE)
- Nueva traducción: KLog está ahora también en galllego. (TNX Fuco Mera)
- Actualizada la versión en alemán. (TNX DL5PD)
- Actualizada la versión en español. (EA4TV).
