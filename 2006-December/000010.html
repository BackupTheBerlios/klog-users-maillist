(Versión en español al final)

I am  happy to announce that the release of 0.4.1 version of KLog is
available.

KLog has now a email list(Klog-users at lists.berlios.de) for discussion
about KLog, wishlist, bug report, ideas, suggestions tips on using, ...

The main language of the list if English.

Subscribe at:

                https://lists.berlios.de/mailman/listinfo/klog-users

From the Changelog:

16-Dic-2006 - 0.4.1

- Bug fixed: The distance and beam is also resetted when the call box is 
deleted.
- Bug fixed: Calls /M and /MM do not count for DXCC.(TNX SM5CNQ)
- Bug fixed: It is possible to filter only for needed entities spots.
- Change: The hamlib signal meter has been removed from the GUI.
- New feature: There is a new tool available to find QSO from which the 
QSL is needed that you have still not sent the QSL. Very useful to fi
nd which QSL you need to send first!
- New feature: The MY_GRIDSQUARE is managed by default by all QSO (saves 
the default if none is entered)

KLog is in English, Spanish, Portuguese and German.

KLog can still have many bugs and other not yet implemented features.
You can download KLog from:

http://jaime.robles.nu/klog
and faster from:

http://developer.berlios.de/projects/klog/

You will find the tar.gz sources and the debian package (debian package
only in my homepage)
You can see some screenshots in the web.

The KLog's new-releases email list is created using the addresses of
people I have previously received emails from about KLog. Feel free to
ask me to remove your address if you don't want to be alerted of new
KLog releases.

Thanks for your help and I hope you find KLog useful.
73

Me alegro de anunciaros que ya está disponible la versión 0.4.1 de
 KLog.

Desde ahora KLog tiene una lista de correo(Klog-users at lists.berlios.de)
para charlar sobre KLog, pedir funcionalidades, avisar de errores,
ideas, sugerencias, trucos sobre cómo usarlo, ...

El lenguaje principal de la lista será inglés.

Subscribete en:

                https://lists.berlios.de/mailman/listinfo/klog-users


Del fichero Changelog vemos los cambios:

16-Dic-2006 - 0.4.1

- Error solucionado: La distancia y la orientación de la antena no se 
reiniciaban cuando se borraba el cuadro del indicativo.
- Error solucionado: Los indicativos /M y /MM no cuentan para el DXCC.
(TNX SM5CNQ)
- Error solucionado: Es posible mostrar únicamente avisos de cluster de 
contactos/entidades necesitadas.
- Cambio: Se ha eliminado el medidor de señal de hamlib del interfaz 
gráfico.
- Nueva funcionalidad: Hay una nueva herramienta que permite buscar los 
QSO cuyas tarjetas necesitas confirmar y a los que no has enviado la 
tarjeta. ¡Muy útil para saber qué tarjetas QSL debes enviar primero!
nd which QSL you need to send first!
- Nueva funcionalidad: Se utiliza el MY_GRIDSQUARE (tu locator actual) 
de forma predeterminada para todos los contactos (guarda el 
predeterminado si no se especifica otro)

KLog está disponible en inglés, español, portugués y alemán.

KLog puede tener aún muchos errores y otras funciones por implementar.
Puedes descargar KLog de:
http://jaime.robles.nu/klog
y más rápido desde:
http://developer.berlios.de/projects/klog/

Encontrarás las fuentes en tar.gz o el paquete de Debian (el paquete de
Debian sólo en mi página)
Puedes ver capturas de pantalla en la web.



La lista de aviso de nuevas versiones de KLog la he ido creando con las
direcciones de las personas que previamente me han enviado algo sobre
KLog. Si prefieres que no te envie estos mensajes, avisame y quitaré tu
dirección de la lista.

Gracias por tu ayuda y espero que KLog te resulte útil.
